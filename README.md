# Korean Spell Checker Workflow for Alfred4

Alfred에서 네이버 맞춤법 검사기를 사용하여 맞춤법 검사를 할 수 있게 해주는 workflow입니다.

## Install workflow
`Korean Spell Checker.workflow`를 다운받아서 실행합니다.

## Usage
```
sc [text]
```
